# Air Pollution Prevention Web Application

## Overview
The Air Pollution Prevention Web Application is a full-stack web platform designed to display air quality data and promote environmental awareness. The system allows users to securely register, manage their accounts, and interact with dynamic environmental data through an interactive and responsive interface.

This project was developed using pure PHP for backend functionality, MySQL for database management, and HTML, CSS, and JavaScript for front-end design.

---

## Features

### User Authentication
- Secure user registration and login system
- Password validation and encrypted credential handling
- Session-based authentication and account management

### Data Management
- Create, retrieve, and display air quality data
- Secure database interaction using PHP and MySQL
- Structured storage and retrieval of environmental information

### User Interface
- Responsive front-end design using HTML, CSS, and JavaScript
- Interactive landing pages with media integration
- Structured layout for improved usability and accessibility

## System Architecture

The application follows a client-server architecture:

- Frontend handles user interaction and data presentation
- Backend PHP processes business logic and database operations
- MySQL manages user data and environmental information

## Installation Guide

### Prerequisites
- XAMPP / WAMP / MAMP
- PHP 7.0+
- MySQL
- Web browser

### Steps

1. Clone the repository "https://github.com/eishuneletha/AirPollutionProject"
2. Move the project folder into your server directory
   -Example for XAMPP:  C:\xampp\htdocs\air-pollution-prevention
3. Configure database connection
  Open: Connect.php
  Update if needed:
  
  ```php
  $host = "localhost";
  $username = "root";
  $password = "";
  $database = "air_pollution_db";
4. Run Create.php to create tables
5. Run the project.
